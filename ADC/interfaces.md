# ADC Interface Specification

## Outputs To MFD

- pitch
- roll
- turn rate
- slip/skid estimate
- `IAS`
- `TAS` optional
- altitude
- `VSI`
- baro setting
- `AOA` optional
- sensor health / stale state

## Configuration Inputs

- baro reference
- pitot/static calibration values
- `AOA` calibration
- alignment offsets
