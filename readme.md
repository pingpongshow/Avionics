This is an Experimental Avionics Project. 

This is a work in progress, looking for contributors. 

Composed of a interconnected system of modules similar to commercial products, all connected through CAN bus 
protocol. 

Modules:
ADC - Air Data Computer (IMU, Airspeed, AOA Static) (scaffolded)
MAG - Magnetometer (Scaffolded)
ADSB - Traffic/FIS-B in (Scaffolded)
GPS ()
MFD - Multi Function display (active work being done)
EIS - Engine Information system ()


This is in no way, shape or for a complete system to date, It SHOULD NOT be used in a aircraft system AS IS
